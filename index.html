Common questions

How do I cancel Tempo+?

On your iPhone: Settings → [Your Name] → Subscriptions → Tempo+ → Cancel Subscription. Cancellation takes effect at the end of your current billing period; the paid features stay active until then. Cancelling Tempo+ never affects the rest of the app — your calendar keeps working exactly as before.

How do I restore a purchase on a new device?

Open Tempo, go to Settings → Restore Purchases, and sign in with the same Apple ID you used to subscribe. The restore button is also on the Tempo+ paywall.

My events aren't showing up. What's wrong?

Tempo reads from the calendar accounts already set up in iOS Settings → Apps → Calendar (or the equivalent for iOS 18+). Check that:

  • Tempo has Full Calendar Access (Settings → Privacy & Security → Calendars → Tempo).
  • The calendars you want are toggled on inside Tempo's Settings → Calendars.
  • The events exist within Tempo's loaded window (±12 months from today).

Does Tempo work with Google Calendar and Outlook?

Yes — Tempo uses Apple's EventKit framework, which reads any calendar account configured in iOS Settings (iCloud, Google, Microsoft Exchange, Outlook.com, Yahoo, CalDAV). Add the account in iOS Settings first; Tempo will see it automatically.

How does the AI assistant work? Is my data private?

When you type a prompt to Tempo's AI features, the prompt text is sent to Anthropic's API to generate a response. We don't log prompts on our infrastructure, don't attach a Tempo-generated user identifier, and don't use the data for any other purpose. Calendar events themselves never leave your device. Full details in our Privacy Policy.

Will my data sync to a second device?

Calendar events sync via whatever account you're using (iCloud, Google, etc.) — Tempo doesn't move them. Tempo's own data — tasks, busy windows, themes, preferences — is mirrored via your personal iCloud Key-Value store when iCloud is signed in, so a second device with the same Apple ID picks up your setup automatically.
